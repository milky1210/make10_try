# Make 10 Challenge

4つの異なる数字（1〜9）をすべて1回ずつ使い、四則演算（`+`, `-`, `*`, `/`）と括弧を駆使して「10」を作るパズルゲームです。

## アプリケーション概要

フロントエンド技術（HTML, CSS, JavaScript）のみで構築されており、GitHub Pagesなどの静的ホスティングサービスで動作します。

### 主な機能・仕様

- **全126問の網羅**: 1〜9の異なる4つの数字の組み合わせ（9C4 = 126通り）が全て出題されます。
- **辞書順出題**: 問題は数字の小さい組み合わせから辞書順（例: `{1, 2, 3, 4}` → `{1, 2, 3, 5}` ...）に出題されます。
- **進捗の自動保存**: 現在何問目を解いているかはブラウザのキャッシュ（LocalStorage）に保存されるため、ページを閉じても続きから再開できます。
- **ゲームループ**: 全126問をクリアすると完了画面が表示され、リセットして最初から遊ぶことができます。
- **多言語対応**: 日本語と英語の表示切り替えに対応しています。

## 技術スタック

- **HTML5**
- **CSS3** (Flexboxによるレスポンシブ配置)
- **JavaScript** (Vanilla JS, LocalStorage API)
